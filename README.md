
---

## UI Library Used

- Bootstrap (via React-Bootstrap)

Bootstrap is used for layout, responsiveness, and pre-styled UI components such as navigation bars, cards, grids, and buttons.

---

## Project Description

The SkillBoost UI is a startup landing page for an online learning platform.  
It includes a navigation bar, hero section, course cards, and footer, all implemented as reusable React components.

---

## Conclusion

This project demonstrates how modern UI libraries can be integrated with React to build clean, scalable, and responsive web applications using component-based architecture.
